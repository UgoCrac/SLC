J'ai fini de concevoir le MCD et j'ai commencer à créer mon API. 
J'ai donc créer un projet symfony avec les composer : 
- Fixtures (Pour créer des faux comptes clients / devis)
- sensio/framework-extra-bundle
- serializer-pack ( Pour recuper un objet et le convertir en JSON ou inverse )
- orm ( doctrine pour communiquer avec la bdd )
- maker-bundle ( Pour créer mes Entitys et controller )

J'ai commencer par changer le lien de ma BDD dans .env puis ai ensuite créé mes entités ( toutes mes tables ), ensuite j'ai mis en place les 
relations pour les tables qui en avaient besoin ( Devis en OneToOne avec Clients, DevisMateriaux en ManyToOne avec Devis et DevisMateriaux
en ManyToOne avec Materiaux )

J'ai ensuite créé des faux clients avec Fixtures ainsi que des faux devis.

J'ai ensuite créé tous les controllers et ai fait toutes les Routes et function pour le CRUD pour l'entité Clients.

J'ai procédé aux test sur Postman :
Requête GET pour récuperer tous les clients
Requête GET pour récuperer un client avec son ID
Requête DELETE pour supprimer un client avec son ID
Requête POST pour créer un nouveaux client
Requête PUT pour mettre à jour un client avec son ID
